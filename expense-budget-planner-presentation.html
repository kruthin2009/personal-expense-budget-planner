01 — Overview
Introduction & Problem

Why this project — and our answer to it

The problem

People find it hard to track where their money goes each day.

Plain spreadsheets are flexible but not visual or interactive.

Most finance apps hide your data inside their own systems.

Our solution

A clean, animated dashboard that is easy to use.

Keeps all data in an open Excel file you fully own.

Best of both worlds: a modern app and transparent data.

02 — Introduction
Goals

Objectives

What the planner is designed to achieve, end to end.

01

Show income vs. expenses clearly using charts and totals.

02

Let users add, edit, and delete transactions easily.

03

Allow a monthly budget per category with progress tracking.

04

06

03 — Objectives
Capabilities

Key Features

Dashboard

Bar, line, and donut charts across 7-day, monthly, and all-time views.

Transactions

Add, edit, and delete entries — double-click any row to edit instantly.

Budgets

Set a limit per category and see how much is used or left, in real time.

Search & Filter

Filter by free text, category, or a custom date range in one place.

Themes

Dark and light mode, plus custom chart colors to match your taste.

Live Excel Sync

Every change is written to expenses.xlsx instantly, no manual export.

04 — Features
Under the hood

Technology Stack

Frontend

React 18 + TypeScript, built with Vite

Styling

Tailwind CSS, Framer Motion, Lucide icons

Charts

Custom SVG charts — bar, line, and donut

Backend

Node.js with Express, a REST API

Storage

Excel workbook via the xlsx library

No database needed —

the Excel file is the database.

One portable file replaces the entire backend store.
